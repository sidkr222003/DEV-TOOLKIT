# Dev Toolkit Milestones

This document tracks the major milestones and roadmap for the Dev Toolkit extension.

## Milestone 1: Initial Release
- Add file size display in status bar
- Add Explorer size decorations
- Add console log removal command
- Publish initial extension version

## Milestone 2: Code Explainer
- Add `src/features/codeExplainer.ts`
- Add `devToolkit.explainSelectedCode` command
- Add `Code Explainer` view panel
- Support selected-code explanation in editor context menu
- Document AI-enabled explanation feature in README

## Milestone 3: Quality & Contribution
- Add branch protection guidance and PR checklist
- Add issue templates for bugs and feature requests
- Improve extension packaging and publish workflow
- Add tests for TS compilation and feature behavior

## Future Enhancements
- Add real-time file coverage insights
- Support configurable explorer decoration thresholds
- Add localization support
- Add automated release notes generation

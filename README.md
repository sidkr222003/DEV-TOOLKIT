# Dev Toolkit

Dev Toolkit bundles 11 essential developer utilities into a single VS Code extension.

## Features

- Quick file notes stored per file with section-specific tags, hover rendering, status bar, and webview management
- Right-click Explorer context menu access for notes, file size, path copy, and folder size
- Console.log cleaner for JavaScript / TypeScript files
- Toggle comments across many languages
- Folder size viewer from the Explorer context menu
- Smart TODO highlighter with configurable keywords
- Reopen last closed file history
- Copy file path formatter with relative, absolute, import, POSIX and alias options
- File size viewer for individual files with human-readable sizing
- Random color generator with HEX, RGB, HSL, pastel, and vibrant colors
- Trailing space cleaner plus optional save-time cleanup
- JSON key sorter with indentation preservation
- Gradient preview hover for CSS and HTML files

## Shortcut Suggestion

You can bind the `Dev Toolkit: Reopen Closed File` command to `ctrl+shift+t` in your keybindings if you want a quick restore shortcut.
